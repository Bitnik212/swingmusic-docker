
# Swingmusic with Filebrowser


## Run this docker-compose

1. Copy/Download [docker-compose.yml](docker-compose.yml)
2. Change ports and folders path
3. Run in same folder:

**Run in foreground:**
```bash 
docker-compose up -d
```
